# 디렉터리 구조

src/  
 ├── 📁 api/ # Axios 인스턴스 및 API 호출 함수  
 │  
 ├── 📁 assets/ # icon, img 등 개발에 필요한 정적 요소  
 │  
 ├── 📁 app/ # Page Routing이나 Layout을 담당  
 │  
 ├── 📁 components # 공통 재사용 UI 컴포넌트  
 │  
 ├── 📁 hooks/ # 커스텀 훅  
 │  
 ├── 📁 contexts/ # 컴포넌트에서 공통으로 알아야 하는 정보  
 │  
 ├── 📁 utils/ # 순수 함수 (날짜 계산, 포맷팅 등 외부 의존성 없는 함수)  
 │  
 ├── 📁 types/ # 전역에서 사용되는 type들의 집합  
 │
├── 📁 constants / # 상수 집합  
 │  
 └── 📁 lib/ # 외부 라이브러리 설정이나 래퍼 함수 (Axios 인스턴스 설정, Firebase 설정 등)

app 내에 컴포넌트 폴더를 두는 방식(co-location)을 택한다.

- 이 때 폴더 앞에는 언더바를 붙인다.
  - `_components`
